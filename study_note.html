Stack

Stack is linear series, the only differences between vector is the stack can acess its top element. The elements in stack follow the last in and first out(LIFO). There are three different basic operation on the stack:

  1. push
  2. pop
  3. top

The basic application of the stack includes

  1. Inversion output: convert binary to digital
  2. Recurrent embedded: problem has self similarity but don't know the depth of recurrent.
